Title:

Screen Time Management App User Departure Prediction and Behavioral Patterns Using Sequence Learning Models

Summary:

This project aims to build a deep learning-based analysis tool that PM can utilize for operational decision-making by utilizing real user data from dopamine management app 'Dopamedi. Based on approximately 200 users, 3000 daily usage records, and 6000 session records, we implement a two-step sequence learning model. The first stage is a GRU-based departure prediction model. By receiving the user's daily usage sequence (app usage time, recommended app usage time, achievement rate, etc.) for the last 14 days, the probability of departure is predicted within the next 14 days. 14 consecutive days of unused is defined as departure The second step is to cluster the departure pattern through a sequence autoencoder. After compressing the sequence immediately before the departure of the defector into a low-dimensional embedding, the departure type is discovered and qualitatively analyzed through KMeans clustering and t-SNE visualization. The core value of this project is that it deals with both supervised and unsupervised representation learning as a single sequence model, and the results will be directly utilized in establishing Dopamedi’s user retention strategy.
