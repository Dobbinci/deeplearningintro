Title:

User Drop-off Prediction and Behavioral Patterns Using Sequence Learning Models

Summary:

This project aims to build a deep learning-based analysis tool that PM can use for operational decision-making, using real-world user data from a screentime management app that I have managed. The dataset contains approximately 200 users, 3,000 daily usage records, and 6,000 session records, and I implement a two-step sequence learning approach.

The first stage is a GRU-based churn prediction model. Using each user's daily usage sequence over the past 14 days (e.g., app usage time, recommended usage time, achievement rate), the model predicts the probability of churn within the next 14 days. I define churn as 14 consecutive days of inactivity.

The second stage clusters churn patterns using a sequence autoencoder. By compressing the usage sequence immediately before churn into a low-dimensional embedding, I identify different churn types and analyze them qualitatively through KMeans clustering and t-SNE visualization.

The key contribution of this project is that it combines supervised prediction and unsupervised representation learning within a unified sequence modeling framework. The results will be directly applied to improve user retention strategy.
